# Roadmap: Hermes Curator Evolver

## Phase 1 — v0.1 Read-only Evidence Plugin

Deliver a usable, tested Hermes plugin that captures safe evidence through hooks and reports it through CLI/tool/slash command paths. No semantic models and no skill mutation.

Plans:
1. `phase-1/PLAN.md` — Implement package, storage, hooks, reports, tests, docs, and release-ready repo.

## Phase 2 — Proposal and Verifier Layer

Add proposal generation from evidence and a verifier gate, still defaulting to dry-run/no-apply.

## Phase 3 — Optional Semantic Candidate Generation

Add embedding/rerank candidate generation as advisory evidence only.

## Phase 4 — Guarded Apply

Add explicit, backed-up, auditable apply flow for verified targeted changes.
